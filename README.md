
# 🍎🍌🥭 Fruit Detection with YOLOv8

This project trains a YOLOv8 model to detect **11 fruit classes** (apple, avocado, blueberry, capsicum, cherry, kiwi, mango, orange, rockmelon, strawberry, wheat).  
It uses the [DeepNIR 11 Fruits Dataset](https://www.kaggle.com/datasets/enddl22/deepnir-11fruits) and runs in **Google Colab**.

---

## 📂 Project Structure
Fruit_detection_YOLO_model/

│── requirements.txt # dependencies
│── notebook.ipynb # Colab notebook (training & testing)
